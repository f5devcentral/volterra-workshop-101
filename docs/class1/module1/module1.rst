Module 1 - Use Case 1 - Publish public application
##################################################

In this section, we will publish an existing and public internet facing application through Volterra Global Network.

The use case is simple:

* An application is already up and running anywhere (Datacenter, public cloud ...)
* This application has a public IP address or FQDN
* We will use and leverage the Volterra Global Network to publish this application
* The application will available everywhere though all Volterra Regional Edge locations
